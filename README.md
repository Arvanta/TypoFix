# TypoFix - مبدل متن فارسی و انگلیسی
# 🔄 مبدل هوشمند چیدمان کیبورد (Keyboard Layout Converter)

آیا تا به حال برایتان پیش آمده که یک متن طولانی را تایپ کنید و بعد از نگاه کردن به صفحه نمایش متوجه شوید که زبان کیبوردتان اشتباه بوده است؟ (مثلاً به جای **سلام** تایپ کرده باشید `sghl`). 

این پروژه یک ابزار تحت وب سبک، سریع و مدرن است که به صورت آنی و هم‌زمان (Real-time)، متونی که با چیدمان کیبورد اشتباه (فارسی به انگلیسی یا انگلیسی به فارسی) تایپ شده‌اند را اصلاح می‌کند.

![description](https://github.com/Arvanta/TypoFix/blob/main/Screenshot.png?raw=true)

## ✨ ویژگی‌های کلیدی
* **تبدیل دوطرفه و آنی:** پردازش و اصلاح متن به محض تایپ کردن، بدون نیاز به فشردن دکمه اضافی.
* **شخصی‌سازی شده و دقیق:** اصلاح دقیق کلیدهای چالش‌برانگیز بر اساس کیبوردهای استاندارد و شخصی (مانند جابجایی دکمه‌های `پ` با `\` و `ژ` با `C`).
* **رابط کاربری:** طراحی کاملاً ریسپانسیو (پاسخگو در موبایل و دسکتاپ) با بهره‌گیری از فونت زیبای Vazirmatn.
* **کپی سریع با یک کلیک:** دارای دکمه اختصاصی کپی خروجی همراه با سیستم اعلان وضعیت (Toast Notification).
* **حریم خصوصی ۱۰۰٪:** تمام پردازش‌ها به صورت کاملاً محلی (Local) در مرورگر شما انجام می‌شود و هیچ داده‌ای به سرور ارسال نخواهد شد.

## 🛠️ تکنولوژی‌های استفاده شده
* HTML5
* CSS3 (همراه با متغیرهای CSS برای مدیریت آسان استایل‌ها)
* JavaScript (Pure / Vanilla JS)
